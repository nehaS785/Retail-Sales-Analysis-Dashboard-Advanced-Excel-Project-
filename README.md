# Retail Sales Analysis Dashboard (Advanced Excel Project)
This project is an end-to-end Advanced Excel–based Sales Analytics Dashboard designed to analyze retail performance using real sales data.It includes data cleaning, KPI generation, and dynamic dashboards to help stakeholders make informed decisions.

## Project Objective
The objective of this project is to analyze retail sales data using Advanced Excel techniques and create an interactive dashboard that helps stakeholders track performance, identify trends, and make informed business decisions. This includes understanding sales patterns, top-performing products, regional performance, and customer behavior.

## Discription
This project involves performing end-to-end sales data analysis using Advanced Excel tools.The dataset includes product details, customer information, sales transactions, regions, and revenue-related attributes.

## Key Features
1.Data Cleaning & Preparation-Removed duplicates, handled missing values, standardized data formats.
2.KPI Calculation using Advanced Excel Functions-SUMIFS, COUNTIFS, VLOOKUP/XLOOKUP, IFERROR, DATE functions, etc.

## Created Interactive Dashboard using:
1.Pivot Tables.
2.Pivot Charts.
3.Slicers (Category, Region, Month filters).
4.Dynamic visuals for better insights.

## Insights Generated
1.Monthly sales trends.2.Top-performing products.3.Region-wise revenue comparison.4.Customer purchase patterns.5.Profit contribution analysis

## Recommendations
1.Focus on High-Performing Products-Increase stock availability and marketing of top-selling items to boost overall revenue.
2.Improve Sales in Low-Performing Regions-Introduce targeted promotions, discounts, or region-specific strategies to increase sales in weak regions.
3.Optimize Inventory Planning-Use monthly trends and product demand patterns to forecast inventory needs and reduce stockouts.
4.Enhance Customer Targeting-Analyze customer purchase behavior to design personalized offers and loyalty programs.
5.Promote Seasonal Best-Sellers-Monthly trend analysis shows high-performing months—use this to plan seasonal campaigns and maximize profit.
